# Underdog Fantasy Front End Challenge - Pokedex

**Technologies**:
`ReactJS`, `PokeAPI`, `CSS`,

**Installation**:

`git clone [this repo]`

`cd /frontend-challenge-underdog`

Run `npm install`

Run `npm start` to start it up

App runs at `localhost:3000`

[PokeAPI](https://pokeapi.co/)
